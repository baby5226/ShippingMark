# 裝箱單產生器 (網頁版)

一個基於網頁的裝箱單產生器，可以將 Excel 檔案轉換為 PDF 格式的裝箱單，支援中文顯示和動態分頁。

## 🌟 功能特色

- **📊 Excel 檔案匯入**：支援 .xlsx 和 .xls 格式
- **� 智慧多表頭支援**：自動偵測同一檔案中的多個區塊 (支援不同標題/表頭)
- **🔢 箱號自動展開**：支援輸入 `1-5` 或 `1~5` 自動展開為連號嘜頭
- **�📄 PDF 產生**：自動產生高品質的 PDF 裝箱單，支援動態分頁
- **🔤 中文支援**：完整支援繁體中文字體顯示 (Noto Sans TC)
- **⚡ 純前端技術**：無需後端伺服器，可直接在瀏覽器中使用

## 📋 使用說明

### 1. 準備 Excel 檔案
您的 Excel 檔案需要包含以下欄位 (位址不固定，認名稱)：

- **板號** (必要)：分組依據，顯示在 PDF 右下角
- **箱號** (必要)：分組依據，顯示在 PDF 右上角 (支援 `1-5` 範圍寫法)
- **貨號**
- **品號**
- **品名**
- **數量** (或 **訂量**)
- **單位**

> **提示**：系統支援 **多區塊** 格式。只要在每個區塊上方保留一列標題，並在板號/箱號列重新定義表頭，程式即可自動識別。

### 2. 使用步驟
1. 點擊「選擇 Excel 檔案」按鈕選擇您的檔案
2. 系統會自動解析並預覽 (若有多個區塊，標題會顯示提示)
3. 確認資料無誤後，點擊「產生 PDF 並預覽」
4. PDF 檔案會在新分頁中開啟，您可以下載或列印

### 3. 匯出範例
點擊「匯出 Excel 範例」可下載標準格式檔案。
預設欄位順序：貨號(A) | 品號(B) | 品名(C) | (空) | (空) | **數量(F)** | **單位(G)** | **板號(H)** | **箱號(I)**

## 🛠️ 技術規格

- **前端框架**：純 HTML5 + CSS3 + JavaScript
- **PDF 產生**：pdf-lib.js
- **Excel 處理**：SheetJS (xlsx.js)
- **字體支援**：NotoSansTC-Regular.ttf

## 📁 檔案結構

```
├── index.html              # 主程式檔案
├── NotoSansTC-Regular.ttf  # 中文字體檔案
├── README.md              # 說明文件
└── .gitignore             # Git 忽略檔案
```

## 🚀 部署方式

### GitHub Pages (推薦)
1. 上傳至 GitHub 儲存庫
2. 設定啟用 GitHub Pages (Source: main branch)
3. 完成！

## ⚠️ 注意事項

- **箱號範圍**：支援 `1-5`, `10~15`, `20 - 25` 等格式。
- **欄位別名**：Excel 若有「訂量」欄位，會自動視為「數量」。
- **字體檔案**：`NotoSansTC-Regular.ttf` 必須與 `index.html` 在同一目錄。
